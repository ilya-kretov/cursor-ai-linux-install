== Overview

This guide provides instructions for installing Cursor AI on Linux systems.

== Prerequisites

* Linux operating system

== Installation

. Download the latest release
. Extract the archive
. Run the installation script

[source,bash]
----
chmod +x ./install.sh
./install.sh
----
